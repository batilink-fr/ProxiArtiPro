Site ProxiArtiPro V2

Contenu :
- Site complet en HTML
- Pages reliées entre elles
- Menu mobile
- Logo intégré si disponible
- Footer commun avec SIRET et email
- Formulaires en version gratuite mailto

Mise en ligne GitHub :
1. Décompresser le ZIP.
2. Envoyer tous les fichiers dans le dépôt GitHub.
3. Vérifier que index.html est bien à la racine.
4. Activer GitHub Pages sur la branche main, dossier /root.

Formulaires automatiques :
Pour recevoir les demandes directement sur Gmail sans ouvrir l'application mail du visiteur, il faudra créer une clé gratuite Web3Forms puis remplacer l'action des formulaires.
